# Разведочный анализ данных (EDA)

## 1. Предобработка данных
### Источники данных
OHLCV: Kaggle dataset 'russia-stocks-prices-ohlcv' (ежедневные данные по российским акциям)
Макро: ЦБ РФ (USD/RUB через XML_dynamic.asp, ключевая ставка через SOAP KeyRateXML, инфляция через Excel UniDbQuery, M2 через Excel monetary_agg.xlsx)
Новости: Kasymkhan/RussianFinancialNews (финансовые новости на русском), сентимент — RuBERT-tiny2

### Шаги предобработки
Загрузка и очистка OHLCV: переименование колонок, парсинг дат, удаление дубликатов
Технические индикаторы: SMA(20,50,200), EMA(12,26), RSI(14), MACD(12,26,9), Bollinger Bands(20), ATR(14), Stochastic(14,3), Williams %R, CCI(20), OBV, Momentum(20), ROC(20), Parkinson volatility(20)
Макро-фичи: загрузка через API ЦБР, forward-fill до ежедневной частоты, расчёт производных (m2_growth_3m, real_rate, usd_rub_vol_20d, usd_rub_dev_from_ma20)
Новостной сентимент: RuBERT-tiny2 для классификации тональности, агрегация по дням (mean, count, std)
Временные фичи: циклические encoding для дня недели/месяца, флаги конца месяца/квартала
Лаги и дельты: 1м/3м лаги для макро, 1д/5д/20д лаги для цен и объёмов
Рыночные фичи: среднерыночная доходность, относительная доходность тикера, rolling beta-like
Внутридневные паттерны: диапазон дня, тени свечи, размер тела, верхняя/нижняя тень
Сигналы пересечения: sma_20_50_cross, sma_50_200_cross, price_above_sma20, macd_bullish

### Обработка пропусков
CatBoost нативно поддерживает NaN — заполнение медианой не требуется (исключение: макро-данные заполняются forward-fill, т.к. инвестор знает последнюю опубликованную ставку)
Пропуски в индикаторах (SMA200, RSI и др.) — естественные границы данных, не заполняются

## 2. Ключевые наблюдения EDA
### Распределения
Лог-доходности имеют тяжёлые хвосты (эксцесс ~3200), не являются нормальными — это отражает реальность рынка с кризисными событиями
Объёмы распределены экспоненциально, есть дни с аномально высокой активностью
Цены имеют огромный диапазон: от копеек (TGKB ~0.006₽) до десятков тысяч (VSMO ~15,360₽) — используем лог-доходности и относительные фичи

### Сезонность
Эффект дня недели: четверг показывает отрицательную среднюю доходность (-0.08%), суббота/воскресенье — положительную (0.37% и 1.14%). Это НЕ артефакт, а отражение гэпа открытия понедельника (carry-through выходных). Оставляем как фичу.
Месячный эффект: январь — лучший месяц (+0.25%, "январский эффект"), май — худший (-0.10%, "Sell in May"). Август тоже сильный (+0.14%).
Квартальный эффект: Q1 лучший (+0.10%), Q2 худший (-0.02%)
Волатильность по дням: воскресенье показывает повышенную волатильность — это артефакт rolling-расчёта, который "видит" предстоящий гэп понедельника. Используем как предиктор волатильности открытия.

### Корреляции с таргетом (target_return_20d)
Топ-2 предиктора: body_size (0.154) и intraday_range (0.152) — внутридневные паттерны сильнее макро
Макро работает через изменения: usd_rub_delta1m (0.092) >> usd_rub (0.041) — динамика важнее уровня
Технические индикаторы: sma_50_200_cross (0.047), rsi (0.043) — тренд и перекупленность значимы
Относительные фичи: relative_log_ret_1d (0.093), relative_return_20d (0.077) — моментум относительно рынка важнее абсолютного

### Макро-влияние
Линейная корреляция макро-уровней с дневной доходностью близка к нулю (USD/RUB: 0.007, Key Rate: -0.0007) — это нормально, макро влияет через лаги, изменения и взаимодействия
Матрица корреляций макро-фичей показывает высокую автокорреляцию уровней, но дельты и отклонения — уникальная информация
m2_growth_3m слабо коррелирует с другими макро — уникальный предиктор

### Новостной сентимент
Средний сентимент слегка негативный (-0.02) — кризисные периоды доминируют в датасете
Распределение мультимодальное: пики при -0.75 (сильно негативные), 0 (нейтральные), +0.5 и +0.9 (позитивные)
Корреляция с дневной доходностью слабая (0.017), но статистически значимая — работает через нелинейные пороги (CatBoost увидит)
Агрегированный сентимент отражает общее рыночное настроение; для тикер-специфичного эффекта требуется NER-извлечение упоминаний компаний

### Таргеты
Классы сбалансированы (~50/50) для всех горизонтов — стандартные метрики адекватны
Дисперсия доходности растёт с горизонтом: σ₁д=5.7% < σ₂₀д=17.5% < σ₂₀₀д=52.5% — соответствует теории √T
Средняя доходность растёт: 1д=0.03% → 200д=7.59% (годовая ~9.6%, реалистично для РФ)
Скошенность: 5д=1.32 (правый хвост, больше прыжков вверх), 200д=0.27 (почти симметрично)

### Качество данных
Пропуски: 78 колонок с пропусками, но все объяснимы (границы истории тикера для индикаторов и таргетов)
Дубликаты (ticker, date): 0 — данные чистые
Выбросы в ценах (3×IQR): 11.4% — это НЕ ошибки, а отражение разницы масштабов между тикерами (от копеек до тысяч). Используем относительные фичи.
Нулевые объёмы: 9,149 (1.2%) — дни без торгов для неликвидных акций. Создаём фичу is_zero_volume.
⚠️ Отрицательные объёмы: обнаружены у тикера TGKA (-1,072) — ошибка данных, требует фильтрации
⚠️ Экстремальные доходности: эксцесс 3217 из-за единичных выбросов. Добавляем фичу is_extreme_move.
Универсум тикеров
Всего 249 тикеров, средняя история 3,078 дней (~12.3 года)
Бимодальное распределение длины: пик у новых тикеров (<500 дней) и основного рынка (3,500–4,500 дней)
Аномалии: GMKN (avg_volume 321M), TRNFP (325M) — возможно ошибка масштаба в исходных CSV
Для горизонта 200д молодые тикеры (<1000 дней) проблематичны — исключаем или используем только общие фичи
