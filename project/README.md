# Итоговый проект по курсу «Инженерия Искусственного Интеллекта»

В этой папке находится итоговый мини-проект по курсу.  
Проект должен демонстрировать применение методов и инструментов инженерии ИИ: работу с данными, модели, пайплайны, сервис, эксперименты и (по возможности) воспроизводимость.

- **Краткое описание (2-4 предложения):**  
Проект решает задачу формирования инвестиционных рекомендаций на российском фондовом рынке. На основе исторических данных о ценах, макроэкономических индикаторов и новостного сентимента. Сервис принимает текущий портфель пользователя и свободные денежные средства, агрегирует прогнозы моделей и выдаёт рекомендации по покупке, продаже или отложенным действиям для краткосрочной и долгосрочной стратегий.

## 2. Структура проекта

Проект организован в следующей структуре:

- `requirements.txt` – зависимости проекта (библиотеки Python, необходимые для запуска).
- `report.md` – отчёт по проекту (постановка задачи, данные, эксперименты, результаты).
- `self-checklist.md` – чеклист самопроверки проекта перед сдачей.
- `notebooks/` – экспериментальные ноутбуки:
  - EDA, прототипы моделей, отдельные эксперименты.
- `src/` – основной код проекта:
  - модули для работы с данными;
  - обучение моделей;
  - запуск сервиса/скриптов.
- `data/` – демонстрационные/учебные данные:
  - **только открытые, синтетические или обезличенные данные;**
  - при необходимости – небольшие выборки для быстрого запуска.
- `configs/` – конфигурационные файлы:
  - `.yaml`, `.json`, `.env.example` и т.п.
- `tests/` – тесты (юнит-тесты, простые проверки).
- `artifacts/` – сохранённые модели, отчёты, результаты экспериментов.

Если структура проекта отличается от базовой, опишите это здесь.

## 5. Данные

OHLCV – ежедневные цены российских акций, открытый датасет Kaggle olegshpagin/russia-stocks-prices-ohlcv.

Макро – ключевая ставка, инфляция, денежная масса M2, курс USD/RUB (источник – ЦБ РФ, данные в коде).

Новости – датасет Kasymkhan/RussianFinancialNews, сентимент рассчитан через rubert-base-cased-sentiment.

Признаки – более 50 технических индикаторов, макроэкономические показатели и их лаги, новостной сентимент, временные метки.

В репозитории хранятся только небольшие примеры (data/processed/combined_features.parquet не включён из‑за размера, для демонстрации можно запустить ноутбук подготовки данных).

Итоговая оценка за проект выставляется по пятибалльной шкале (2-5).

  - минимальный функционал есть;
  - по чеклисту `self-checklist.md` выполнено **меньше 5** пунктов;
  - эксперименты и документация слабо проработаны.
- **4** – хороший, рабочий проект:
  - сервис запускается по `README.md`, `/predict` использует реальную модель;
  - есть данные, EDA и эксперименты с метриками;
  - структура кода и конфигураций в целом адекватна;
  - по чеклисту выполнено **не менее 5** пунктов.
- **5** – сильный, хорошо проработанный проект:
  - аккуратно реализован сервис и пайплайн;
